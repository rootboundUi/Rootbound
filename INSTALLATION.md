# Rootbound Theme - Installation & Setup Guide

## Quick Start (3 Steps)

### 1. Upload Theme to Shopify
```
1. Go to Admin â†’ Online Store â†’ Themes
2. Click "Add theme" â†’ "Upload ZIP file"
3. Select Rootbound folder and wait for upload
```

### 2. Customize Theme Settings
```
1. On your theme, click "Customize"
2. Go to "Theme settings" in left sidebar
3. Update:
   - Colors (Primary, Secondary, Accent)
   - Typography
   - Site Navigation
   - Social Media Links
   - Newsletter settings
4. Click "Save"
```

### 3. Configure Home Page
```
1. Still in "Customize" view
2. Click sections to edit:
   - Hero Banner (image, text, button)
   - Featured Products
   - Featured Collections
   - Testimonials
   - Gallery
3. Publish when done
```

## Section Customization

### Hero Banner
**What it does**: Large background image with centered text and CTA

**How to customize**:
1. Under "Customize" â†’ Find "Hero Banner" section
2. Upload background image
3. Edit title, subtitle, button text
4. Set button link (e.g., `/collections/all`)
5. Choose text alignment

### Featured Collections
**What it does**: Display 3-5 collection cards

**How to customize**:
1. Under "Hero Banner" section
2. Click "Add block"
3. Select collection from dropdown
4. (Optional) Upload collection image preview
5. Repeat for more collections

### Featured Products
**What it does**: Showcase 8 products from selected collection

**How to customize**:
1. Select collection from dropdown
2. Title and subtitle auto-populated
3. Choose number of products (3-12)
4. Toggle "View All" button

### Testimonials
**What it does**: Customer review cards with ratings

**How to customize**:
1. Click "Add block" for each testimonial
2. Enter customer quote
3. Add customer name and location
4. Upload customer avatar image
5. All ratings show 5 stars

### Image Gallery
**What it does**: Grid of lifestyle/community images

**How to customize**:
1. Click "Add block" for each image
2. Upload lifestyle photo
3. (Optional) Add caption
4. Select grid columns (2, 3, or 4)

## Product Setup

### Required Fields
- **Title** - Product name
- **Description** - Product details (shows on product page)
- **Image** - Main product image
- **Price** - Selling price

### Recommended Fields
- **Compare at price** - Original price (shows sale badge)
- **Variants** - Size, color options
- **Tags** - For filtering
- **Vendor** - Brand/collection name
- **Collections** - Organize products

### Image Best Practices
- Use square or landscape images
- Upload multiple images (enables gallery)
- Optimize for web (72 DPI, < 100KB per image)
- Use descriptive alt text

## Performance Tips

1. **Optimize Images**
   - Use JPG for photos (smaller file size)
   - Use PNG for graphics (better quality)
   - Resize to actual display size
   - Use Shopify's built-in image compression

2. **Clean Up Unused Files**
   - Remove old hero images
   - Delete test products
   - Uninstall unused apps

3. **Monitor Performance**
   - Use Google PageSpeed Insights
   - Check Core Web Vitals
   - Test on 4G mobile connection

## Troubleshooting

### Theme won't upload
- Ensure ZIP file is the correct format
- Check file isn't corrupted
- Try different browser
- Contact Shopify support

### Sections not appearing
- Refresh the Customize page
- Clear browser cache
- Try incognito window
- Restart browser

### Products not displaying
- Ensure products are published
- Check collection assignments
- Verify product images uploaded
- Review product visibility settings

### Mobile menu doesn't work
- Clear browser cache
- Check JavaScript is enabled
- Test in new incognito window
- Try different browser

### Images not loading
- Check image files uploaded
- Verify correct image path
- Try JPG instead of PNG
- Reupload image
